# GPA Calculator
A Next.js app to calculate GPA, deployed on Vercel.

## Features
- Add courses with grades/credits
- Calculate GPA
- Save to LocalStorage
- Styled with CSS

## Setup
1. Clone repo
2. Run `npm install`
3. Run `npm run dev`

## Deployment
[gpa-calculator.vercel.app](https://gpa-calculator-rust.vercel.app/)
